# IP测试工具 (IPTest)

这是一个基于Flet框架开发的跨平台IP测试工具，提供简单直观的图形界面来进行IP连通性测试。

## 功能特点

- 简洁美观的图形用户界面
- 支持IP地址连通性测试
- 实时显示测试结果
- 跨平台支持（Windows、macOS、Linux）

## 安装要求

- Python 3.7 或更高版本
- 依赖包（见 requirements.txt）

## 安装步骤

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

运行以下命令启动程序：
```bash
python main.py
```

在输入框中输入要测试的IP地址，点击"开始测试"按钮即可开始测试。

## 注意事项

- 某些系统可能需要管理员权限才能执行ping操作
- 建议使用Python虚拟环境运行本程序
